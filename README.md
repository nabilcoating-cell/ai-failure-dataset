# AI Failure Dataset

This project contains structured AI failure cases extracted from real research papers.

## Goal
Help developers understand real-world AI risks such as:
- Prompt Injection
- Backdoor Attacks
- Bias
- Autonomous Risk

## Example Case

**Title:** ClawSafety: "Safe" LLMs, Unsafe Agents  
**Failure Type:** Prompt Injection  
**Risk Level:** HIGH  

**Scenario:**  
A malicious input can manipulate an AI agent  

**Impact:**  
Data leakage or unintended actions  

---

## Why this project?

I’m exploring whether structured failure data can help developers build safer AI systems.
